It is a website that goes over PFAS levels in water within the UK. 
It allows users to check their loacations, and see the levels of 
PFAS regionally.
Additionally the app goes over the correlation between PFAS and bladder
cancer cases within the UK. 
Users can also gain education on PFAS.

This website runs out of app.py as a html link.
